# Frontend Boilerplate (React)

Minimal React + TypeScript + Vite starter with Tailwind CSS and code quality tooling.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS

## Scripts

```bash
npm install
npm run dev       # dev server
npm run build     # production build
npm run lint      # ESLint
npm run format    # Prettier
```

## Structure

```
src/
  App.tsx
  main.tsx
  index.css
  assets/
```

## Breakpoints

Defined in `tailwind.config.ts`: `xs` 320 · `sm` 425 · `md` 768 · `lg` 1024 · `xl` 1440 · `2xl` 2550
